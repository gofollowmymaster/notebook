# 小程序环境配置



## 配置开发环境

> 1. node  安装    官网下载
> 2. npm 插件安装  npm  install
> 3. 小程序代码编译构建 
>    -    npm  build  watch
>    - wepy框架  用  wepy build --watch 
> 4. 导入编辑器
> 5. Appid为小程序唯一Appid  在readme总有
> 6. 申请成为该小程序开发者
> 7. 修改配置不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书
> 8. 修改后端域名(ip) 通常在utils/server.js
